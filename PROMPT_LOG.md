# Prompt Log

This document records how generative AI tools were used during the creation of the project.

## Use of AI

AI was used as a development assistant during the design and implementation of the website.

The primary uses included:

- generating layout structures for HTML pages
- refining CSS styling and animation systems
- debugging JavaScript for scroll animations
- improving clarity and structure of written descriptions

## Human Direction

All conceptual ideas, structure of the exhibition, and overall theme of the project were created by the author.

AI tools were used primarily to assist with:

- code troubleshooting
- layout refinement
- wording improvements

The final design decisions and conceptual framing were determined by the author.

## Ethical Use

AI was used as a tool to accelerate development and assist with debugging, not to replace the conceptual design process.

All final content was reviewed and edited to ensure it accurately represented the intended project concept.
